# Bookmark Manager

### User Stories

Complete:
```
As a user,
So that I can use the links I have created,
I'd like to view my list of links.

As a user,
So that I can expand my list of links with new and interesting ones,
I'd like to be able to add links with a title.

As a user,
So that I can refer to and organize my links,
I'd like to be able to add subject tags to links.

As a user,
So that Ican find specific links,
I'd like to filter links by tags.

As a user,
So that I can start using the bookmark manager,
I'd like to sign up.

As a user,
So that I can access my bookmark manager,
I'd like to sign in.

As a user,
So that I can leave my bookmark manager safely,
I'd like to sign out.

As a user,
So that I can still access my bookmarks if I forget my password,
I'd like to be able to reset it securely.
```

Incomplete:
```
As a user,
So that I can discard unwanted links,
I'd like to be able to delete links.

As a user,
So that I can refer to and organize my links,
I'd like to be able un-tag them.
```
